Dark Alley Admin is used to manage Dark Alley content repositories.

Source

Source content status

Returns details about the content source.

path Parameters
owner
required
string

Name of the source owner.

path
required
string

Path to the source content.

Responses

https://admin.da.live/source/{owner}/{path}

Response samples

  • 200
Content type
application/json

    },
  • "aem": {
